# Response Contract

统一交付格式，避免“会做但不可复现”。

## 成功响应模板

1. 执行命令：
   - `<完整命令>`
2. 关键结果：
   - `task_id: ...`
   - `model_id: ...`
   - `output: ...`（如果有输出文件）
3. 结果摘要：
   - `<一句话描述>`

## 失败响应模板

1. 失败命令：
   - `<完整命令>`
2. 错误摘要：
   - `<简短错误>`
3. 可执行重试命令：
   - `<retry command>`
4. 已记录失败样本：
   - `~/.vox/agent/failures.jsonl`

## 输出偏好

1. 默认优先 `--json` 或 `--format ndjson`。
2. 仅在用户明确要求时输出富文本说明。
